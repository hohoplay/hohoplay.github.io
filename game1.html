캐릭터 선택

🏹궁수 호야
🧙마법사 호야
🎮

기본 조작 (1/4)

🏹
이동 — ↑↓←→ 방향키로 호야를 움직입니다.이동 중에도 타이핑 가능합니다!
⌨️
타이핑 공격 — 몬스터 머리 위 단어를 입력하면 공격합니다.가장 가까운 몬스터의 단어가 자동으로 매칭됩니다.
🎯
직업 선택 — 궁수 호야(🏹)와 마법사 호야(🧙) 중 선택!궁수는 이동속도 빠름 / 마법사는 투사체 속도 느리지만 화려함
❤️
체력 — 시작 시 HP 3칸, 레벨업 성장으로 최대 12칸까지 확장!HP가 0이 되면 모험 실패. 몬스터에게 닿지 않도록 조심하세요!
⌨️

기본 조작 (2/4)

🔤
한/영 전환 — 반드시 한글 모드로 입력하세요.화면 좌측 하단 KO/EN 표시로 현재 모드 확인 가능합니다.
⎋
입력 초기화 — ESC로 현재 입력을 즉시 지울 수 있습니다.오타가 났을 때 빠르게 초기화하세요. 1초 후 자동 초기화됩니다.
⚡
스킬 — Z 보호막(100코인) / X 범위 공격(500코인)코인을 모아 위기 상황에 사용하세요!
💡

기본 조작 (3/4)

📊
성장 현황 — P키 또는 ⭐ LV 클릭으로 성장 상태 확인.레벨업 중 몬스터는 완전 정지! 천천히 카드를 골라요.
🔢
글자 수 전략 — 1글자는 빠르지만 보상 적고, 3글자는 느리지만 보상 큽니다.3글자 단어를 3번 연속 완성하면 콤보 보너스 +10XP·🪙×10!
🌸
4계절 변화 — 웨이브가 진행될수록 봄→여름→가을→겨울로 변합니다.배경, 나무, 꽃, 풀이 계절에 맞게 바뀝니다!
🗺️

기본 조작 (4/4)

🗺️
미니맵 — 우측 상단 레이더로 주변 상황을 파악하세요.🔵 플레이어 / 🔺 비석 / ⬜ 일반몬스터 / 🟠 불버섯 / 🔴 보스
⚠️
월드 축소 — 레벨업마다 맵 경계가 좁아집니다!경계 밖으로 나가면 강제 복귀됩니다. 비석을 빠르게 정화하세요.
🏆
클리어 조건 — 12웨이브 보스를 모두 처치하면 전설의 수호자!웨이브가 높아질수록 BGM이 빨라지고 난이도가 올라갑니다.
🍄

몬스터

👁️
일반 버섯 — 1~3글자 단어를 타이핑하면 처치됩니다.1글자=1XP·1코인 / 2글자=2XP·2코인 / 3글자=3XP·3코인 드롭
🔥
불버섯 — 1분마다 등장하는 강적! HP가 🔥🔥🔥 3개입니다.3글자 단어를 3번 완성해야 처치! 처치 시 +10XP·10코인 드롭
👑
보스 버섯 — 비석 16개 정화 & LV.6 달성 시 출현!5글자 단어를 여러 번 입력해야 합니다. 처치하면 다음 웨이브 진행.
💀
몬스터 충돌 — 몬스터에게 닿으면 HP 1칸 감소 + 넉백!보호막(Z)이 있으면 데미지 1회 차단합니다.
🗿

비석 & 코인

🗿
고대 비석 — 맵에 16개가 배치됩니다.비석 단어를 입력하면 속담 시험이 시작됩니다. 띄어쓰기 포함 정확히 입력!
📜
속담 시험 — 비석 정화 중 몬스터는 일시 정지됩니다.속담을 정확히 입력하면 비석이 정화되고 경험치를 획득합니다.
🪙
코인 수집 — 맵에 자동으로 코인이 뿌려집니다.플레이어가 코인 위를 걸어가면 자동 수집! 몬스터 처치 시에도 드롭됩니다.
📦
보물 상자 — 맵에 숨겨진 상자를 찾아 단어를 입력하세요.처치 시 +20코인을 즉시 획득합니다.
✨

성장 시스템

⭐
레벨업 — 경험치가 가득 차면 성장 카드 3개 중 1개를 선택!레벨업 중 모든 몬스터가 일시 정지됩니다. 천천히 골라요.
💪
8가지 성장 항목 — 재물의 운 / 불굴의 생명 / 광역 사격 / 현자의 지혜냉기 오라 / 신속의 발걸음 / 폭발 투사체 / 황금 자석 (각 LV.12)
🍀
행운을 빕니다! — 호야와 함께 버섯 군단을 물리쳐 주세요.어려우면 Z/X 스킬을 적극 활용하세요!
